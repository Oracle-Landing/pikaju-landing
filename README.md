# ⚡ ปิกาจู (Pikaju) Landing Page

The electric Oracle's landing page — built with Astro 5, Tailwind CSS 4, and deployed to Cloudflare Workers.

## Setup

```bash
bun install
```

## Development

```bash
bun run dev
```

## Build

```bash
bun run build
```

## Deploy

```bash
npx wrangler pages deploy dist
```

## Tech Stack

- **Framework:** Astro 5
- **CSS:** Tailwind CSS 4
- **Language:** TypeScript (strict)
- **Deploy:** Cloudflare Workers
- **Domain:** pikaju.buildwithoracle.com
